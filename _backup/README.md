# EARN WITH NAZO

## Overview
**EARN WITH NAZO** is a web application project designed to provide a platform for courses and earning opportunities. This repository contains the initial skeleton structure for the application.

## Project Structure
The project follows a standard MVC (Model-View-Controller) architectural pattern:

```
EARN WITH NAZO/
├── public/             # Publicly accessible files
│   └── assets/         # Static assets (images, CSS, JS)
└── src/                # Source code
    ├── Config/         # Configuration files
    ├── Controllers/    # Application logic and request handling
    ├── Models/         # Data structure and database interactions
    └── Views/          # User interface templates
        ├── auth/       # Authentication related views
        ├── courses/    # Course related views
        ├── dashboard/  # User dashboard views
        └── layouts/    # Shared layout templates
```
